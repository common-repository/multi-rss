Multi RSS combines and displays multiple RSS feeds as a single feed. The plugin depends on the SimplePie RSS Feed Parser for PHP.

== Description ==

Muli RSS leverages the SimplePie in order to combine feeds of all sorts (RSS, ATOM, XML). The feeds are aggregated and then soretd by date and time for display. Feeds can easily be added or removed.
	
== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
Add the shortcode [multiRSSDisplay] where you want it to display.
